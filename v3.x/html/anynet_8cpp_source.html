00028 /*anynet
00029  *
00030  *Network setup file format
00031  *example 1:
00032  *router 0 router 1 15 router 2
00033  *
00034  *Router 0 is connect to router 1 with a 15-cycle channel, and router 0 is connected to
00035  * router 2 with a 1-cycle channel, the channels latency are unidirectional, so channel 
00036  * from router 1 back to router 0 is only single-cycle because it was not specified
00037  *
00038  *example 2:
00039  *router 0 node 0 node 1 5 node 2 5
00040  *
00041  *Router 0 is directly connected to node 0-2. Channel latency is 5cycles for 1 and 2. In 
00042  * this case the latency specification is bidirectional, the injeciton and ejection lat
00043  * for node 1 and 2 are 5-cycle
00044  *
00045  *other notes:
00046  *
00047  *Router and node numbers must be sequential starting with 0
00048  *Credit channel latency follows the channel latency, even though it travels in revse
00049  * direction this might not be desired
00050  *
00051  */
